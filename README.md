## Overview
This repository contains the code and artifacts to reproduce the fake-job-detection project:
- rule-based label generation (`src/label_generation.py`)
- feature engineering (`src/feature_engineering.py`)
- model training (`src/train_models.py`)
- evaluation and visualization (`src/evaluate.py`)
- notebooks for EDA and experiments

> NOTE: For privacy and size reasons the full dataset is not committed. See `data/README_DATA.md` for instructions.
